## 功能
- [x] webpack相关配置（可以在vue.config.js自行开启）
- [x] 语言国际化
- [x] 动态路由（公用路由及私有路由）
- [x] 主题切换（less）
- [x] ...

### 下载依赖
```
cnpm install
```
### 运行
```
cnpm run serve
```
### 打包
```
cnpm run build
```
