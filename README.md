## 简介

基于vue-cli4+webpack 封装，便于下次新项目快速构建一个完备的模板项目
（包含架手架引入、过滤器封装、ajax封装、路由引入封装、代理配置调试、less预编译配置引入、less主题切换、语言国际化、网页加载进度条）

## 功能
- [x] webpack相关配置（可以在vue.config.js自行开启）
- [x] 语言国际化
- [x] 动态路由（公用路由及私有路由）
- [x] 主题切换（less）
- [x] 网页加载进度条（nprogress）
- [x] ...

### 下载依赖
```
cnpm install
```
### 运行
```
cnpm run serve
```
### 打包
```
cnpm run build
```
